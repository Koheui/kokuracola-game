# 🎮 武蔵がくる — 小倉コーラCMゲーム

「広告はゲームへ進化する」実証のための、小倉コーラ プロモーションゲーム。
ベルトスクロールアクション(参考: カプコン「天地を喰らう」)。

## 遊び方

`index.html` をブラウザで開くだけで動きます(ビルド不要・依存ゼロ)。
Webページへの掲載は、フォルダごとサーバーに置いて `<iframe src="index.html">` で埋め込むのが簡単です。

ローカルサーバーで動かす場合:

```bash
cd kokuracola_game
python3 -m http.server 8765
# → http://localhost:8765
```

## 操作

| 操作 | PC | スマホ |
|---|---|---|
| 移動(8方向) | 矢印キー / WASD | 左下の十字パッド |
| ジャンプ | X / K / スペース | 右下・上段ボタン |
| 攻撃(3連コンボ) | Z / J | 右下・中段ボタン |
| 必殺技「巌流旋風斬り」 | C / L | 右下・下段ボタン |
| 決定 | Enter | タップ |

- 敵を倒すと**巌流ゲージ**が溜まり、MAXで必殺技が使えます
- **小倉コーラ**を取ると一定時間、攻撃力2倍+攻撃が派手になります
- 娘娘の肉焼飯=大回復、辻利のグリーンティー=小回復
- 体力ゲージ制・残機なし。力尽きてもその場でコンティニュー可能

## キャラクターを本物の絵にする(重要)

リファレンス(天地を喰らう)水準のキャラクターは、スプライトシートPNGを
`assets/img/sprites/` に置くことで実現します。**シートの仕様・AI生成用プロンプト例は
[docs/asset_spec.md](docs/asset_spec.md) を参照**。置くだけでコード描画から自動で切り替わります。

## 画像の差し替え

現在の絵はすべてコード生成です。`assets/img/` に以下の名前でPNGを置くと、
**コード変更なしで自動的に差し替わります**。

| ファイル | 内容 | 推奨サイズ |
|---|---|---|
| `mihagino.png` | ステージ1背景(三萩野) | 1920×540 |
| `city.png` | ステージ2背景(市街地) | 1920×540 |
| `castle.png` | ステージ3背景(小倉城) | 1920×540 |
| `cola.png` | 小倉コーラのボトル | 48×96(縦長・背景透過) |
| `cola_logo.png` | 小倉コーラのロゴ | 400×160(背景透過) |
| `niku.png` | 娘娘の肉焼飯 | 48×48(背景透過) |
| `tea.png` | 辻利のグリーンティー | 48×48(背景透過) |

背景は横に自動タイル(繰り返し)されます。継ぎ目が目立たない絵にしてください。

## ファイル構成

```
index.html        エントリーポイント
css/style.css     レイアウトとタッチUI
js/assets.js      画像生成と差し替え読み込み
js/audio.js       効果音・BGM(WebAudio生成、外部ファイル不要)
js/input.js       キーボード+タッチ入力
js/entities.js    プレイヤー・敵・アイテム・エフェクト
js/game.js        ステージ進行・カメラ・HUD・画面遷移
docs/             仕様書
```
